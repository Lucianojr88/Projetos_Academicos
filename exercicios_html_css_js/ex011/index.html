Borda Animada

Aprendi a criar uma borda animada utilizando a propriedade personalizada CSS (@property), onde é possível definir novas propriedades CSS que têm um comportamento específico, semelhante às propriedades nativas do CSS.


Posso aplicar alguns tipos de dados através do valor 'syntax:', como: <'length'>, <'color'>, <'number'>, <'angle'>, etc.


É preciso definir o seu valor inicial através do valor 'initial-value:', se haverá herança do container pai ou não através do valor 'inherits:'
